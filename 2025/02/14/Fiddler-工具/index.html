Web调试工具Fiddler

测试主要用来抓包，修改前端请求参数以及模拟后端返回，也可以做弱网测试。

工作原理

Fiddler是以代理服务器的形式工作的。所谓代理，就是在客户端和服务器之间的桥梁，对于客户端，代理服务器充当服务器的角色，负责接受请求并返回响应数据，对于服务器，它充当客户端的角色，负责转发请求并接收返回数据。

代理模式有什么用呢？它能在不改变原始对象的情况下，为其添加额外的功能。

常用功能

抓取指定IP地址的包

使用过滤器，在Hosts标签下第二个下拉列表中选择 Show only the following Hosts，填写IP即可。如果想要去掉css、js等包，勾选Request Headers标签下的Hide if URL Contains，设置完成后点击IP框右上角保存。

抓取HTTPS协议包

默认不能抓HTTPS包，在Tools-Options-HTTPS标签中勾选抓取HTTPS连接和解密HTTPS传输即可。

抓取手机APP包

Tools-Options-Connections中开启允许远程连接。

手机WIFI设置代理为Fiddler所在主机IP，端口默认为8888。手机访问这个IP：8888，下载并安装证书，完成。
