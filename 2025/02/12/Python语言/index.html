Python学习笔记（卷一）

一、数据类型

​ Python语言中，常用的数据类型有七类：

  1. 整数
  2. 小数
  3. 字符串
  4. 列表
  5. 元组
  6. 字典
  7. 自定义数据类型

回忆一下，Java的数据类型，分为基本数据类型和引用数据类型。

​ 基本数据类型：六种数字类型（byte、short、int、long、double、float）、布尔类型（boolean）、字符类型（char）；

​ 引用数据类型：类、接口、数组；

​ 另外，像是String、Enum枚举类型都是基于引用数据类型来实现的。

1、数字对象

​ Python的数字对象比较简单，包括整数、浮点数（小数）。

​ 在做运算时，数字运算符号优先级可以看官网。Python运算符优先级。

字符串方法

  • count

count 方法可以返回字符串对象包含了多少个参数指定的字符串

  • find

find 方法 会在字符串中查找 参数 子字符串，并返回该 参数字符串 在其中第一个出现的位置索引

如果字符串对象里面 没有 要查找的参数字符串，就会返回 -1

  • split、splitlines

split 方法以参数字符串为分割符 ，将字符串 切割为多个 字符串，作为元素存入一个列表，并返回这个列表。

splitlines是把字符串 按换行符 进行切割，比如

  • join

join 是将列表中的字符串元素 以某字符串为连接符， 连接 为一个字符串

  • strip 、 lstrip 、 rstrip

strip方法可以将 字符串前面和后面的空格删除，但是不会删除字符串中间的空格， lstrip 删除左边空格、 rstrip删除右边。

  • replace

用来 替换 字符串里面 所有指定的 子字符串 为另一个 字符串

  • startswith 和 endswith

startswith 方法检查字符串是否以参数指定的字符串 开头，如果是，返回True，否则返回False

endswith 方法检查字符串是否以指定的字符串 结尾，如果是，返回True，否则返回False

  • isdigit

isdigit 方法检查字符串是否全部由数字构成，如果是，返回True，否则返回False

  • 字符串的倒序

3、列表

可以把列表想象为 一连串的储物格，其中每个储物格里面都可以存储 任何类型 的对象，包括另一个列表。

列表页是有sequnce特性的，也支持切片操作。

列表中元素的内容是可以变化的，甚至替换的数量也可以不同。

合并两个列表用加号+。

列表方法

  • append

append方法会在列表后面添加一个元素。

注意，append 方法的返回值是 None，而不是新的列表对象。

  • insert

在 指定位置插入一个元素，同样的，insert方法的返回值也是 None

  • pop

从列表 取出并删除 一个元素

  • remove

remove是根据元素的值进行删除的，remove从第1个元素开始，寻找 和参数对象相等的元素，如果找到了，就删除，找到后，不会继续往后寻找其它相等的元素。

  • reverse

reverse方法将列表元素倒过来，同样的， reverse 方法的返回值也是 None

  • index

index 方法返回参数对象在列表中的位置，也就是索引

  • sort

可以使用 sort 方法对列表进行排序。

调用sort 方法，列表中的元素一般都是数字或者字符串。

sort方法的返回值是None。

列表推导式

效果：把一个列表里面的每个元素，经过相同的处理，生成另一个列表。
